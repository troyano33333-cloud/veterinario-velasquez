1Resumen del Proyecto

El proyecto es un sitio web profesional de una sola página (SPA) diseñado para el Dr. Germán Velásquez Gutiérrez, Médico Veterinario Internista con servicio a domicilio en Medellín, Colombia. El objetivo principal es captar y convertir clientes potenciales mediante una experiencia visual atractiva, clara y funcional, que dirija a los usuarios hacia WhatsApp para agendar consultas.

El sitio combina colores cálidos, animaciones fluidas y un diseño moderno y profesional apto para negocios de salud animal.

🎨

Estilo Visual

Disney-inspired. Colores cálidos, animaciones suaves, personajes SVG únicos.

📱

100% Responsivo

Adaptado para móvil, tablet y escritorio con 3 breakpoints CSS.

⚡

Sin Backend

Frontend puro. HTML + CSS + JS vanilla. Sin frameworks ni servidor.

6Sistema de Modal de Servicios

Hay un único modal reutilizable que se rellena dinámicamente con JavaScript al hacer clic en cualquier botón "Más info". No hay 8 modales separados — solo uno que cambia su contenido.

Flujo del Modal

Clic en "Más info" de un servicio
→
openModal('id') llamado
→
JS busca en objeto servicios
Rellena: emoji, título, desc, lista, precio
→
Asigna color al header
→
Muestra overlay con display:flex

Estructura del Objeto servicios

PropiedadTipoDescripción
colorString (hex)Color del header del modal. Único por servicio.
emojiString (emoji)Icono grande en el encabezado del modal.
tituloStringNombre del servicio. Se muestra como título principal.
descStringPárrafo descriptivo del servicio (2-3 oraciones).
puntosArray<String>Lista de 4 puntos clave con checkmarks.
precioStringPrecio "Desde $XX.000" mostrado en caja naranja.

9Módulos JavaScript

El JavaScript está completamente modularizado en la carpeta js/. Cada archivo tiene una sola responsabilidad y exporta su función de inicialización. El punto de entrada es main.js, que los importa todos usando ES Modules nativos (type="module").

1

🐾 Paws Particles — Huellas Animadas

Genera dinámicamente 40 elementos div.paw-particle con posición aleatoria (0–100% horizontal), tamaño 12–28px, duración de animación 10–22s y delay 0–20s. La animación CSS pawFall los hace caer desde arriba con rotación suave.

2

🧭 Navbar — Scroll, Toggle Móvil y Enlace Activo

Listener de scroll agrega clase .scrolled al navbar cuando el usuario baja más de 20px (cambia fondo de transparente a blanco). Toggle de menú hamburguesa para móvil. Cierre automático del menú al hacer clic en un enlace. Enlace activo naranja: detecta qué sección está visible en el viewport y aplica clase .active al enlace correspondiente — con estilo de pastilla naranja sólida (background: var(--primary)). No afecta al botón CTA de «Contacto».

3

✨ Scroll Reveal — Animaciones de Entrada

Usa IntersectionObserver con threshold 0.1 para detectar elementos que entran en viewport. Agrega clases .reveal, .reveal-left, .reveal-right que CSS convierte en animaciones de fade+slide. Delays escalonados para grids.

4

🗺️ Leaflet Map — Mapa de Medellín

Inicializa Leaflet.js con tiles CartoDB Voyager centrado en Medellín (6.2442, -75.5812), zoom 12. Marcador personalizado (huellita rotada 45°) con popup de contacto. Círculo de cobertura 10km radio en naranja semitransparente. Envuelto en try/catch.

5

🎠 Testimonials Slider — Carrusel Automático

Slider con auto-advance cada 5 segundos. Calcula el ancho de cada tarjeta dinámicamente para el transform. Crea dots indicadores dinámicamente. Botones prev/next con lógica circular (vuelve al inicio al llegar al final).

6

📬 Contact Form — WhatsApp Integration

Al hacer submit previene el comportamiento por defecto. Lee todos los campos (nombre, teléfono, mascota, servicio, mensaje) y construye un mensaje de texto formateado. Abre wa.me/573001234567?text=... con window.open. Muestra notificación de éxito.

7

🪟 Modal de Servicios — js/modal.js

Contiene el objeto SERVICIOS con los 8 servicios (nombre, descripción, precio, ícono). Las funciones openModal(id) y closeModal() están encapsuladas en este módulo. Los botones de la sección usan data-service="id" y la detección se hace por delegación de eventos en el documento. Cierre con clic externo o tecla Escape.

8

🔢 Counters — Animación de Estadísticas

Anima los números del hero (+500 mascotas atendidas, 14 años de experiencia) con efecto suavizado cuando el usuario llega a esa sección. Usa IntersectionObserver para disparar la animación una sola vez. Cada contador tiene duración y paso configurables.

11Diseño Responsivo

El sitio usa 3 breakpoints con media queries en css/responsive.css para adaptarse a todos los dispositivos.

Breakpoint Ancho Cambios principales
Desktop > 768px Layout de escritorio completo. Grids de 3 columnas. Navbar horizontal.
Tablet ≤ 768px Servicios → 2 columnas. Casos → 2 columnas. Navbar → hamburguesa. About → 1 columna. Sección padding 70px.
Móvil ≤ 480px Servicios → 1 columna. Casos → 1 columna. Botones hero → columna. Hero título → 2rem. Stats hero → columna. About frame → 100% ancho.
Mobile-first en botones: Los perros y gatos SVG del hero se ocultan en mobile (display: none) para no sobrecargar la pantalla. El mapa se adapta al 100% del ancho disponible.
